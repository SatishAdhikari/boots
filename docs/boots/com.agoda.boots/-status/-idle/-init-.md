# &lt;init&gt;

`Idle()`

Idle status means that [bootable](../../-bootable/index.md) hasn't been touched yet by the boot system.
All freshly added [bootables](../../-bootable/index.md) through [add()](../../-boots/add.md) get the idle status automatically.
